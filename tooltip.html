Toolbar

A jQuery plugin that creates contextual tooltip style toolbars



Toolbar allows you to quickly create tooltip style toolbars for use in web applications and websites. The toolbar is easily customisable using the twitter bootstrap icons and provides flexability around the toolbars display and number of icons. Toolbar is actively being developed to continue to provide a great user experience.

Examples

Horizontal toolbar

JS 
$('#normal-button').tooltip({
	content: '#user-options', 
	position: 'top'

Usage

Include the plugin javascript file along with jquery

<script src="js/jquery.min.js"></script>
<script src="js/jquery.tooltip.js"></script>

Include the css files

<link href="css/tooltips.css" rel="stylesheet" />
<link href="css/bootstrap.css" rel="stylesheet" />

Define your toolbar HTML

<div id="user-options">
	<a href="#"><i class="icon-lock"></i></a>
	<a href="#"><i class="icon-play-circle"></i></a>
	<a href="#"><i class="icon-text-width"></i></a>
	<a href="#"><i class="icon-inbox"></i></a>
	<a href="#"><i class="icon-tasks"></i></a>
</div>

Attach the toolbar to a button/link, whatever you want really

$('#normal-button').tooltip(options);

Options

  • Content

    The id of the div containing the icons.

  • Position

    Indicate the position of the toobar relative to the element its attached agaisnt. Select either 'Top' or 'Left'.
